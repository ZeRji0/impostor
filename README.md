# Impostor — Juego de palabras (web móvil)
Página web para jugar en un único móvil. Muestra la palabra al resto y el rol de impostor a uno solo.

## Cómo usar
1) Abre `index.html` en tu móvil (o súbelo a GitHub Pages / Netlify / Vercel).  
2) Elige número de jugadores, (opcional) nombres.  
3) Selecciona **Mis palabras** (escribe una lista separada por comas) o **Aleatorias**.  
4) Pulsa **Comenzar ronda** y pasa el móvil: cada jugador verá su palabra o si es impostor.  
5) Cuando termine el reparto, pulsa **¡Todos listos!** y empezad a jugar.

## Despliegue rápido
- **GitHub Pages:** sube los 3 archivos al repositorio y activa Pages en Settings → Pages (branch `main`, folder `/`).  
- **Netlify / Vercel:** arrastra la carpeta y listo.

## Personalización
- Cambia la lista de palabras por defecto en `index.html` (objeto `PALABRAS_BASE`).  
- Ajusta estilos en `styles.css`.
