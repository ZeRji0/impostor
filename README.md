# Impostor — Juego de palabras (ES con palabras aleatorias reales)
- **Aleatorias (ES):** obtiene una palabra real desde Internet en cada ronda. Si falla, avisa “No hay conexión”.
- **Mis palabras:** lista manual separada por comas.
- Al finalizar el reparto, aparece la pantalla **¡Jugad!** con botón **Nueva partida** que reinicia la ronda con los mismos jugadores.

## Cómo usar
1) Abre `index.html` (ideal: subir a GitHub Pages / Netlify / Vercel para mejor soporte JS).  
2) Configura jugadores y el modo de palabras.  
3) Comienza la ronda y pasa el móvil de jugador en jugador.  
4) Al terminar, pulsa **Nueva partida** para otra palabra e impostor nuevos.
